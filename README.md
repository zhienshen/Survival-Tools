# Minecraft 实用生存工具模组 \| Utility Tools Mod

## 📖 模组介绍

一款轻量化、高适配的 Minecraft 生存向实用工具模组，专注于补齐原版生存的工具短板，优化玩家的生存体验。模组无冗余花哨内容，全部功能围绕**原版生存日常需求**设计，贴合原版游戏风格，不破坏生存平衡，适配休闲生存、硬核生存等各类游玩场景。

原版 Minecraft 部分工具功能单一、实用性有限，本模组针对性添加一系列高频实用工具，解决生存过程中采集、探索、收纳、赶路、打怪等场景下的痛点，让生存流程更流畅、便捷，同时最大程度保留原版游戏的核心乐趣。

## 🔧 模组核心功能

本模组新增多款生存刚需实用工具，覆盖生存全场景需求，核心内容如下：

- **高效采集工具**：适配矿石、树木、方块采集的进阶工具，优化采集效率，支持范围采集、快速破块等功能，告别单调重复的挖矿砍树操作，节省大量生存时间。

- **探索辅助工具**：新增探险专用道具，可辅助玩家快速定位资源、标记点位、夜视探索、水下呼吸等，完美适配洞穴探索、下界探险、远古城市探索等高危场景。

- **生存便民工具**：包含便携收纳、快速合成、耐久修复、经验存储等实用道具，解决背包不足、合成繁琐、工具易损耗等常见生存难题。

- **简易战斗辅助工具**：轻量化防御、增伤、续航类工具，平衡原版战斗机制，仅提升游玩体验，不制作超模装备，兼顾趣味性与生存挑战性。

## 📥 安装方式

1. 确保已安装对应游戏版本的 Fabric 加载器；

2. 下载本模组对应版本的 `\.jar` 文件；

3. 将模组文件放入游戏根目录的 `mods` 文件夹；

4. 重启游戏即可正常加载使用。
